# lamboshe-jv-repo
JavaScript Fundamentals Project
📌 Description

This project covers the fundamentals of JavaScript, including variables, data types, functions, control flow, arrays, and objects. It also includes two interactive applications that demonstrate how JavaScript can be used to process and manipulate data.

🚀 Features
1. Interactive Calculator
Perform basic arithmetic operations:
Addition
Subtraction
Multiplication
Division
Handles invalid input and division by zero
2. Student Grade Tracker
Add student names and scores
Store data using arrays and objects
Display student list dynamically
Calculate and display average score
📚 Concepts Covered
Variables and data types
Functions
Conditional statements (if/else)
Loops
Arrays and array methods
Objects
Basic DOM manipulation
